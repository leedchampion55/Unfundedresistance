UFR 1BP / 1BN EA

Automates First Bar Positive / First Bar Negative breakouts with mechanical entries, OCO order handling and symbol-specific trailing logic.

  • Automatic 1BP / 1BN logic — detects the first session candle and places orders above/below range.
  • OCO orders — once one side triggers, the other cancels automatically.
  • Symbol-specific trailing — tune trails per index or pair.
  • Flexible SL/TP — SL from trigger, custom pip amounts, adjustable RR.
  • Dynamic lot sizing — fixed risk, risk per point, or % balance.
  • Execution control — set your preferred candle/time window.
